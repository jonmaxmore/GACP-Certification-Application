## 1️⃣ QC OFFICER (Quality Control Officer)

### 🎯 **บทบาท**: เจ้าหน้าที่ตรวจสอบคุณภาพเอกสารเบื้องต้น

### 📊 **Level**: Junior / Entry-level
- ไม่จำเป็นต้องมีความรู้ GACP ลึกซึ้ง
- Training 1-2 สัปดาห์ ทำงานได้
- เงินเดือน: ฿18,000 - 25,000

### 🔍 **หน้าที่หลัก**: ตรวจ**รูปแบบ**และ**ความครบถ้วน** (Form & Completeness)

#### **ตรวจอะไร?** (Checklist-based)

**1. Document Quality Check (คุณภาพเอกสาร)**
```
[ ] ไฟล์เปิดได้หรือไม่ (ไม่ corrupted)
[ ] ขนาดไฟล์เหมาะสม (ไม่ใหญ่หรือเล็กเกินไป)
[ ] รูปภาพชัด/อ่านได้ (ไม่เบลอ)
[ ] ไม่ใช่ไฟล์ผิดประเภท (ต้อง PDF/JPG)
```

**2. Document Completeness (ความครบถ้วน)**
```
ต้องมี 8 เอกสาร:
[ ] 1. สำเนาบัตรประชาชน (เจ้าของฟาร์ม)
[ ] 2. สำเนาทะเบียนบ้าน
[ ] 3. แผนที่ฟาร์ม (GPS Coordinates)
[ ] 4. ใบอนุญาตปลูก (จาก อบต./กรมแพทย์แผนไทย)
[ ] 5. รายงานตรวจสอบดิน (Soil Test)
[ ] 6. แผนผังฟาร์ม (Layout)
[ ] 7. รูปถ่ายฟาร์ม (ขั้นต่ำ 5 รูป)
[ ] 8. ประวัติการปลูก (ถ้ามี)

ถ้าขาด → ปฏิเสธทันที + แจ้งเกษตรกร
```

**3. Basic Information Check (ข้อมูลพื้นฐาน)**
```
[ ] ชื่อ-นามสกุลตรงทุกเอกสารหรือไม่
[ ] เลขบัตรปชช.ตรงหรือไม่
[ ] ที่อยู่ฟาร์มตรงกับแผนที่หรือไม่
[ ] ขนาดพื้นที่ (ต้องระบุเป็นไร่/งาน/ตร.ว.)
[ ] วันที่ส่งใบสมัคร (ไม่เกิน 30 วัน)
```

**4. Date & Signature Check (วันที่และลายเซ็น)**
```
[ ] เอกสารทุกหน้ามีลายเซ็น
[ ] วันที่ไม่หมดอายุ (Soil test < 6 เดือน)
[ ] ใบอนุญาตยังใช้งานได้
[ ] วันที่สอดคล้องกัน (ไม่มีอนาคต)
```

**5. File Format & Size (รูปแบบไฟล์)**
```
[ ] PDF: ไม่เกิน 10MB ต่อไฟล์
[ ] รูปภาพ: JPG/PNG ไม่เกิน 5MB ต่อรูป
[ ] รวมทั้งหมดไม่เกิน 50MB
```

### ⏱️ **เวลาในการตรวจ**
- เคสง่าย (ครบถ้วน): **5-10 นาที**
- เคสปกติ: **15-20 นาที**
- เคสซับซ้อน (ข้อมูลไม่ครบ): **30 นาที**

### 📊 **Output / การให้คะแนน**

QC Officer ให้ **Pre-Score** (0-100):

```javascript
Score Calculation:
- เอกสารครบ 8 ชิ้น: 40 คะแนน
- รูปภาพชัด อ่านได้: 20 คะแนน
- ข้อมูลถูกต้อง ไม่มี error: 20 คะแนน
- วันที่/ลายเซ็นครบ: 10 คะแนน
- File format ถูกต้อง: 10 คะแนน
────────────────────────────
Total: 100 คะแนน

Result:
- 90-100: ✅ PASS → ส่งต่อ Reviewer (Fast Track)
- 70-89:  ⚠️ PASS → ส่งต่อ Reviewer (Normal)
- 50-69:  ❌ WARNING → ส่งต่อ Senior Reviewer
- 0-49:   ❌ REJECT → ส่งกลับเกษตรกร (ขาดเอกสาร)
```

### 🚫 **สิ่งที่ QC Officer ไม่ทำ**

❌ **ไม่ประเมินเนื้อหา** (Content Assessment)
- ไม่ตัดสินว่าฟาร์มผ่านมาตรฐาน GACP หรือไม่
- ไม่วิเคราะห์ความเสี่ยง
- ไม่ดูประวัติเกษตรกร
- ไม่อ่านรายละเอียดทางเทคนิค

❌ **ไม่ตัดสินใจอนุมัติ/ปฏิเสธ**
- แค่ตรวจว่าเอกสารครบหรือไม่
- ตัดสินใจให้ Reviewer เท่านั้น

❌ **ไม่ติดต่อเกษตรกร**
- ไม่โทร / ไม่อีเมล
- ระบบจะแจ้งเตือนอัตโนมัติ

### 📈 **KPI**
- จำนวน applications ตรวจต่อวัน: **15-20 ใบ**
- ความแม่นยำ: **> 95%** (ไม่พลาดเอกสารที่ขาด)
- เวลาเฉลี่ย: **< 20 นาทีต่อใบ**

### 💼 **Reporting Line**
- Report to: **Team Lead QC / Reviewer Manager**
- Work with: QC Team (3 คน)

## 2️⃣ REVIEWER (Document Reviewer)

### 🎯 **บทบาท**: ผู้ตรวจสอบและประเมินเอกสารเชิงลึก

### 📊 **Level**: Senior / Expert-level
- ต้องมีความรู้ GACP standards
- เข้าใจกฎหมายและระเบียบ
- ประสบการณ์ 2-3 ปีขึ้นไป
- เงินเดือน: ฿35,000 - 50,000

### 🔍 **หน้าที่หลัก**: ประเมิน**เนื้อหา**และ**คุณสมบัติ** (Content & Eligibility)

#### **ตรวจอะไร?** (Expert Judgment)

**1. GACP Eligibility Assessment (คุณสมบัติตามมาตรฐาน GACP)**
```
✅ ฟาร์มอยู่ในพื้นที่ที่เหมาะสม (ไม่ใกล้โรงงาน/ถนนใหญ่)
✅ แหล่งน้ำสะอาด (ไม่เป็นมลพิษ)
✅ ดินเหมาะสมกับการปลูก (Soil test ผ่าน)
✅ ขนาดพื้นที่เหมาะสม (> 1 ไร่)
✅ มีแผนจัดการฟาร์ม (Farm Management Plan)
```

**2. Farmer Background Check (ประวัติเกษตรกร)**
```
✅ ไม่มีประวัติการละเมิดกฎหมาย
✅ ไม่ถูกแบน/blacklist
✅ ไม่เคยถูกเพิกถอนใบรับรอง
✅ ประวัติการทำฟาร์ม (ถ้ามี → บวก)
✅ การฝึกอบรม GACP (ถ้ามี → บวก)
```

**3. Risk Analysis (วิเคราะห์ความเสี่ยง)**
```
Risk Factors:
❌ High Risk:
   - ฟาร์มใหม่ (ไม่มีประสบการณ์)
   - ใกล้แหล่งน้ำเสีย
   - พื้นที่มีประวัติปลูกพืชที่ใช้สารเคมีสูง
   
⚠️ Medium Risk:
   - เคยมีใบรับรองแต่หมดอายุ
   - ขนาดฟาร์มใหญ่เกินไป (> 20 ไร่)
   
✅ Low Risk:
   - มีประสบการณ์ปลูก > 3 ปี
   - เคยผ่านการตรวจ GACP มาก่อน
   - Soil test ดี
```

**4. Technical Document Review (ตรวจเอกสารทางเทคนิค)**
```
✅ Soil Test Report:
   - pH level (6.0-7.5)
   - Organic matter > 2%
   - ไม่มีโลหะหนัก
   
✅ Water Source Analysis:
   - แหล่งน้ำสะอาด
   - ไม่มีสารเคมี
   
✅ Farm Layout:
   - มีระบบระบายน้ำ
   - มีพื้นที่แยกปลูก/เก็บเกี่ยว
   - มีโรงเก็บอุปกรณ์
```

**5. Cross-Check with Database (ตรวจสอบข้อมูลซ้ำซ้อน)**
```
✅ ฟาร์มไม่ซ้ำในระบบ (GPS Coordinates)
✅ เกษตรกรไม่ได้สมัครซ้ำ
✅ ที่ดินไม่มีข้อพิพาท
```

### ⏱️ **เวลาในการตรวจ**
- เคสง่าย (Fast Track จาก QC): **20-30 นาที**
- เคสปกติ: **45-60 นาที**
- เคสซับซ้อน (High Risk): **1-2 ชั่วโมง**

### 📊 **Output / การตัดสินใจ**

Reviewer ให้ **Final Decision**:

```javascript
Decision Matrix:

✅ APPROVE (อนุมัติ):
   - QC Score: > 85
   - GACP Eligible: ✅
   - Low Risk: ✅
   - Technical Docs: ✅
   → Action: ส่งต่อ Inspector
   
⚠️ REQUEST INFO (ขอข้อมูลเพิ่ม):
   - เอกสารไม่ชัดเจน
   - ต้องการหลักฐานเพิ่ม
   - คำอธิบายไม่พอ
   → Action: ส่งกลับเกษตรกร (submission count +1)
   
❌ REJECT (ปฏิเสธ):
   - ไม่ผ่านเกณฑ์ GACP
   - High Risk ที่ไม่สามารถแก้ไขได้
   - เอกสารปลอม
   → Action: ปฏิเสธถาวร หรือ ให้แก้ไข (ถ้าเป็นครั้งที่ 3 = ชำระเพิ่ม 5,000฿)
```

### 🚀 **สิ่งที่ Reviewer ทำเพิ่ม**

✅ **Priority Assignment**
```
High Priority:
- ฟาร์มที่มีประสบการณ์ดี
- Score สูง (> 90)
- เอกสารครบถ้วน

Low Priority:
- ฟาร์มใหม่
- Score ต่ำ
- ต้องขอข้อมูลเพิ่ม
```

✅ **Inspector Recommendation**
```
แนะนำประเภทการตรวจ:
- Fast Track (Video Call only) → ฟาร์มคะแนนสูง
- Normal (Hybrid) → ฟาร์มปกติ
- Full Inspection → ฟาร์มเสี่ยงสูง
```

### 🚫 **สิ่งที่ Reviewer ไม่ทำ**

❌ **ไม่ตรวจเอกสารพื้นฐาน**
- QC ทำแล้ว → ไม่ต้องเช็คซ้ำ
- Focus เฉพาะเนื้อหา

❌ **ไม่ลงพื้นที่ตรวจฟาร์ม**
- Inspector เท่านั้นที่ลงพื้นที่
- Reviewer ตัดสินจากเอกสาร

❌ **ไม่อนุมัติใบรับรองขั้นสุดท้าย**
- แค่อนุมัติให้ตรวจฟาร์ม
- Approver เท่านั้นที่ออกใบรับรอง

### 📈 **KPI**
- จำนวน reviews ต่อวัน: **8-12 ใบ**
- ความแม่นยำ: **> 98%** (ไม่ผ่านฟาร์มที่ไม่ควรผ่าน)
- เวลาเฉลี่ย: **< 1 ชั่วโมงต่อใบ**

### 💼 **Reporting Line**
- Report to: **DTAM Manager**
- Work with: QC Team, Inspector Team

## 📊 QC vs Reviewer: Side-by-Side Comparison

| หัวข้อ | QC OFFICER | REVIEWER |
|--------|------------|----------|
| **Level** | Junior (Entry-level) | Senior (Expert) |
| **เงินเดือน** | ฿18,000 - 25,000 | ฿35,000 - 50,000 |
| **ความรู้ GACP** | ไม่จำเป็น (Training 1-2 สัปดาห์) | จำเป็น (ประสบการณ์ 2-3 ปี) |

| **Work Volume** | 15-20 ใบ/วัน | 8-12 ใบ/วัน |

## 3️⃣ INSPECTOR (Farm Inspector)

### 🎯 **บทบาท**: ผู้ตรวจสอบฟาร์มในพื้นที่จริง

### 📊 **Level**: Expert / Field Specialist
- ต้องมีความรู้ GACP มาตรฐาน
- ประสบการณ์ลงพื้นที่
- มีรถยนต์/มอเตอร์ไซค์
- เงินเดือน: ฿30,000 - 45,000 + ค่าเดินทาง

### 🔍 **หน้าที่หลัก**: ตรวจสอบฟาร์มจริง (Onsite) หรือ Video Call

---

### ⚡ **สำคัญ! Inspector มี 3 Modes:**

```
┌─────────────────────────────────────────────┐
│ Mode 1: VIDEO CALL ONLY (Fast Track)       │
├─────────────────────────────────────────────┤
│ เงื่อนไข:                                    │

│ - เคยได้รับรองมาก่อน                        │
│ - Low Risk                                  │
│                                             │
│ กระบวนการ:                                   │
│ 1. นัด Video Call กับเกษตรกร                │
│ 2. ให้เกษตรกรพาดูฟาร์มผ่านกล้อง             │
│ 3. ถามคำถามพื้นฐาน                          │
│ 4. Screen capture เป็นหลักฐาน               │
│                                             │
│ เวลา: 2 ชั่วโมง                             │
│ ค่าใช้จ่าย: ฿500                            │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ Mode 2: HYBRID (Video + Onsite if needed)  │
├─────────────────────────────────────────────┤
│ เงื่อนไข:                                    │
│ - Reviewer Score 70-89                      │
│ - Medium Risk                               │
│                                             │
│ กระบวนการ:                                   │
│ 1. Video Call ก่อน (2 ชั่วโมง)             │
│ 2. ถ้าพบสิ่งผิดปกติ → ลงพื้นที่            │
│ 3. ถ้าไม่มีปัญหา → จบที่ Video Call         │
│                                             │
│ เวลา: 2-6 ชั่วโมง                           │
│ ค่าใช้จ่าย: ฿500-3,000                      │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│ Mode 3: FULL ONSITE (Complete Inspection)  │
├─────────────────────────────────────────────┤
│ เงื่อนไข:                                    │
│ - Reviewer Score < 70                       │
│ - High Risk                                 │
│ - ฟาร์มใหม่ (ไม่เคยได้รับรอง)               │
│                                             │
│ กระบวนการ:                                   │
│ 1. ลงพื้นที่เต็มรูปแบบ                       │
│ 2. ตรวจ GACP Checklist 50+ รายการ           │
│ 3. ถ่ายรูป 20-30 รูป                        │
│ 4. เก็บตัวอย่างดิน/น้ำ (ถ้าจำเป็น)          │
│ 5. สัมภาษณ์เกษตรกร                          │
│                                             │
│ เวลา: 1 วัน (6-8 ชั่วโมง)                   │
│ ค่าใช้จ่าย: ฿3,000-5,000                    │
└─────────────────────────────────────────────┘
```

### 📋 **GACP Inspection Checklist (50+ Items)**

**Category 1: ที่ตั้งฟาร์ม (Location) - 10 คะแนน**
```
[ ] ห่างจากโรงงาน/แหล่งมลพิษ > 500 เมตร
[ ] ไม่อยู่ใกล้ถนนใหญ่ (ฝุ่น/เสียง)
[ ] แหล่งน้ำสะอาด
[ ] ดินเหมาะสม (ไม่เป็นหิน/ทราย)
[ ] ไม่อยู่พื้นที่น้ำท่วมซ้ำ
```

**Category 2: แปลงปลูก (Cultivation Area) - 15 คะแนน**
```
[ ] มีรั้วกั้นครอบคลุมทั้งหมด
[ ] ป้ายชื่อฟาร์มชัดเจน
[ ] แบ่งโซนปลูกชัดเจน
[ ] ระบบระบายน้ำดี
```
