## 🧪 Conditional Documents

เอกสารเหล่านี้อาจถูกเพิ่มโดยอัตโนมัติตามข้อมูลที่ผู้ใช้กรอก:

| เงื่อนไข | เอกสารที่เพิ่ม |
|---------|---------------|
| กรณีสูญหาย (Lost) | สำเนาใบแจ้งความ |
| กรณีชำรุด (Damaged) | รูปถ่ายใบรับรองที่ชำรุด |
| สถานะ: จดแจ้ง (Notify) | ใบรับจดแจ้ง |
| สถานะ: มีใบอนุญาต (Licensed) | สำเนาใบอนุญาต BhT |
| มี CCTV | ผังการติดตั้งกล้องวงจรปิด |
| มีหัว/เหง้า (Tuber/Rhizome) | ผลวิเคราะห์สารหนู |
| แหล่งที่มา: ซื้อ (Buy) | ใบเสร็จรับเงินค่าเมล็ดพันธุ์ |
| แหล่งที่มา: นำเข้า (Import) | ใบอนุญาตนำเข้า |
